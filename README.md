# npp-tourism
A mod that removes most* hostile enemies from most** of N++'s solo levels.

*Not including Thwumps or Shove Thwumps, since removing them might make certain levels unbeatable

**Not including ?! or !? tab levels

`tourist.zip` keeps level names intact, `tourist-fae.zip` removes level names.

## Backup
Before installing this mod it's recommended that you backup your save and level data for easy uninstallation.

Save data:
1. Navigate to your save directory.
- Windows: `%USERPROFILE%\Documents\Metanet\N++`
- Mac: `~/Documents/Metanet/N++`
- Linux/Steam Deck: `~/.local/share/Metanet/N++`
- Proton: `~/.steam/steam/steamapps/compatdata/230270/pfx/drive_c/users/steamuser/Documents/Metanet/N++`
2. Copy `nprofile.gz` to a safe location.

Level data:
1. Navigate to the game's directory. This can be done by right-clicking the game on Steam and clicking Manage->Browse local files.
2. Copy `[GAME DIRECTORY]/NPP/Levels` to a safe location.

## Installation
1. Navigate to the game's directory. This can be done by right-clicking the game on Steam and clicking Manage->Browse local files.
2. Copy the contents of `tourist.zip`/`tourist-fae.zip` to `[GAME DIRECTORY]/NPP/Levels`. Overwrite files if asked.

To uninstall, copy your backup `nprofile.gz` file and `Levels` directory to their original locations.
